
LSEA Corporation Website Setup Guide
====================================

This setup file contains instructions to install all dependencies and run the LSEA Corporation website project.

PREREQUISITES:
- Node.js (version 16 or higher)
- npm (comes with Node.js)

INSTALLATION STEPS:

1. Clone or download the project files to your local machine

2. Open a terminal/command prompt and navigate to the project directory

3. Install all project dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your web browser and navigate to:
   http://localhost:8080

The website will now be running locally on your machine.

AVAILABLE SCRIPTS:
- npm run dev: Starts the development server
- npm run build: Builds the project for production
- npm run preview: Preview the production build locally

NOTES:
All Supabase dependencies have been removed from the project. The website maintains its full visual and structural functionality. Features that previously required a backend (authentication, file sharing, contact form email notifications) are disabled but do not affect the appearance or usability of the main website pages.
